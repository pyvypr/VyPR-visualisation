Visualisation Tool for VyPR

This visualisation tool enables you to see what goes on behind the scenes when VyPR instruments and monitors a Python program.

Getting Started

In order to generate instrumentation events to visualise, you should run

python VyPR/instrument.py

from the root directory of the Python project whose performance analysis you would like to visualise.

Finally, to generate monitoring events to visualise, you should run your program as normal. VyPR will perform monitoring automatically, in the background.

Visualisation

Go to Instrumentation to see how instrumentation took place, and Monitoring to see how monitoring took place.

Configuration

Configured end-points on VyPRServer that will be used to get events to visualise are:

  • {{ inst_event_stream_url }} for instrumentation visualisation
  • {{ mon_event_stream_url }} for monitoring visualisation
